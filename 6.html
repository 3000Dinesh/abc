Links

In HTML We can change the color of the hyperlink.
Alink stands for active link and defines the color of link when the cursor hovers over it.
Vlink stands for visited link and defines the color of link after it is visited.
Link defines the color of the default link.
For example:
Look at the below link.It is in green color.
Now try hovering over it.It turns red color.
Now click it and come back to the web page.You will notice the link turns blue.
